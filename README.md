## Guidelines
- 
- Add tests relevant to a component or service.

## Useful commands

### Install the dependencies

```
yarn
```

### Run

```
yarn start
yarn build
yarn test
```

### Deploy

_Requires push access, use carefully._

```
yarn deploy
```
