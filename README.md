## Guidelines
* Add tests relevant to a component or service.

## Useful commands

### Install the dependencies
```
yarn 
```

### Run
```
yarn start
yarn build
yarn test
```

### Deploy

*Requires push access, use carefully.*

```
yarn deploy
```
